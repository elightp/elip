1 Introduction

1.1 Organization of ELIP

This specification is split into seven chapters:

  • Chapter 1 - Introduction
  • Chapter 2 - Network connection
  • Chapter 3 - ELIP Packet format
  • Chapter 4 - ELIP Packets
  • Chapter 5 - Operational behavior
  • Chapter 6 - Security

1.2 Terminology

The key words "MUST", "MUST NOT", "REQUIRED", "SHALL", "SHALL NOT", "SHOULD", "SHOULD NOT", "RECOMMENDED", "MAY", and "OPTIONAL" in this specification are to be interpreted as described in IETF RFC 2119 [RFC2119].

Network Connection:

A construct provided by the underlying transport protocol that is being used by ELIP to send ELIP packets.

  • It connects the Client to the Server.
  • It must provides the means to send an ordered, lossless, stream of bytes in both directions.
